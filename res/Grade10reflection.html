Hammer

For this project, a functional mallet/hammer was manufactured. The hammer head was machined on a ProtoTrak milling machine and a tapped hole. The rest of the hammer, the handle, was manufactured on a manual lathe. The handle contains a thread which screws into the tapped hole on the head. The handle features a knurled grip, which makes the hammer more functional. The hammer head contains a radius on each side to prevent denting the edges when hammering, and for a more pleasing look. This operation was done using a radius end mill. The hammer was fully made out of steel, so it has increased driving force. The hammer is approximately 10.375” in length and the handle is approximately 0.875” in diameter.

Threading Project

For the project, a bolt and a nut were created that screw together to prove our knowledge on how to use the manual lathe. For this project, we learned how to thread on a manual lathe. This required looking at the threading dial, and ensuring that the same number/position was used each time. Reaction time is important when threading on a manual lathe to ensure the feed does not crash into the head of the bolt/screw. Multiple passes were taken, and the final thread was tested. The nut had a similar thread, but it was an ID (Inside a diameter) thread. This thread followed the same procedure, but required an ID threading tool instead of an OD. The entire bolt is approximately 3.5” in total length, and has an approximate bolt length of 1.650” across flats.

Simple Cylinder

The simple cylinder project consists of four outside diameters with varying sizes. The axial tolerance of the diameters were not that tight, but the diameter was very tight. This taught us to set a zero on a lathe very precisely. For this precision, it required making very accurate measurements with micrometers before turning the diameters. As imperfections in the machine and issues with repeatability occurred, the X diameter measure was checked often with the reading on the DRO.
